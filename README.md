# Portfolio - John Doe

Portfolio website d'un développeur web et designer, services, projects, and skills.

## Prérequis

- Node.js 18+
- npm ou yarn

## Installation

```bash
npm install
```

## Lancement

### Développement

```bash
npm run dev
```

Le site est accessible à [http://localhost:3000](http://localhost:3000)

### Production

```bash
npm run build
npm start
```

## Linting

```bash
npm run lint
```
